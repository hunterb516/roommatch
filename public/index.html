Roommate Assignment Project

Manage dorm buildings, rooms, and student assignments for incoming freshmen.

Add Student Assign Room View Assignments View All Rooms Find Rooms for Student Find Roommates Building Summary Report

Use the links above to perform each of the seven core functions:

  1. Add Student – Insert a new freshman into the database.
  2. Assign Room – Match a student to a dorm room based on preferences.
  3. View Assignments – See which students are in which rooms.
  4. View All Rooms – List every room, showing availability.
  5. Find Rooms for Student – Find open rooms that meet a student’s needs.
  6. Find Roommates – See other students with identical preferences.
  7. Building Summary Report – Get per‑building and campus‑wide availability stats.
